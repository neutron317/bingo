# ゲーム仕様

## ゲームの流れ

```
1. ホストが「ゲーム作成」→ ルームIDが発行される
2. ホストがURLを参加者に共有
3. 参加者がURLにアクセス → 名前を入力してビンゴカードが配布される
4. ホストがゲームを開始する
5. ホストが番号を引く → 全員のカードにリアルタイム反映
6. ビンゴになった参加者が申告 → ホストに通知
7. ホストが確認・ゲーム終了
```

## ゲーム設定

ホストがゲーム作成時に設定できる。未指定の場合はデフォルト値を使用。

| 設定項目 | デフォルト | 備考 |
|----------|-----------|------|
| カードサイズ | 5×5 | 2〜100の整数で指定可能 |
| 番号の範囲 | 1〜75 | min・max ともに整数 |
| カード生成 | ランダム自動生成 | ホストが手入力モードに切り替え可能 |
| ホストパスワード | なし | 任意で設定可能 |
| 入室パスワード | なし | 任意で設定可能 |

## カードサイズとフリースペース

- カードサイズは 2〜100 の整数で指定する
- **奇数サイズ**（3、5、7 など）: 中央の1マスがフリースペース
- **偶数サイズ**（4、6、8 など）: 中央の 2×2（4マス）がフリースペース
- **サイズ 2 のみ例外**: フリースペースなし（4マスすべて番号で埋める）

## ルール

- ビンゴ判定は縦・横・斜めすべて有効
- 参加時に名前入力を強制する
- 最大参加人数は20人
- ルームはゲーム終了から24時間で自動削除
