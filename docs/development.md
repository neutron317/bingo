# 開発手順

## Phase 1：環境セットアップ
1. 実装用ブランチを切る
2. バックエンドのプロジェクト作成（Hono + TypeScript）
3. Docker Compose で Redis を立ち上げる

## Phase 2：バックエンド実装
4. Redis 接続の確認
5. REST API 実装（`POST /rooms`、`GET /rooms/:id`）
6. WebSocket イベント実装（ホスト・プレイヤー接続、番号を引く、ビンゴ申告など）
7. 再接続・状態復元の実装
8. ルーム有効期限（TTL）の実装

## Phase 3：フロントエンド実装
9. フロントエンドのプロジェクト作成（Vite + React + TypeScript）
10. ルーム作成画面
11. 参加画面（名前入力・パスワード入力）
12. ゲーム画面（カード表示・番号表示）
13. ホスト操作画面（番号を引く・ビンゴ確認）

## Phase 4：結合・デプロイ
14. フロントとバックエンドの結合確認
15. デプロイ（Railway など）
